# Setup Checklist

Use this checklist to track your progress through the setup process.

## Initial Setup
- [ ] Node.js installed (v18+)
- [ ] Git installed
- [ ] GitHub account created
- [ ] Vercel account created
- [ ] Supabase account created

## Project Setup
- [ ] Git repository initialized
- [ ] Next.js project created
- [ ] Next.js running on localhost:3000

## Frontend Setup
- [ ] Tailwind CSS verified working
- [ ] shadcn/ui initialized
- [ ] shadcn/ui components installed (button, card, input, label, badge)
- [ ] Environment variables configured (.env.local)

## Supabase Setup
- [ ] Supabase project created
- [ ] Database connection string copied
- [ ] API keys copied (anon public and service_role)
- [ ] Database host, password saved

## Strapi Setup
- [ ] Strapi installed in /strapi directory
- [ ] PostgreSQL client installed (pg)
- [ ] Strapi database.ts configured
- [ ] Strapi .env file created with all values
- [ ] Secure keys generated for APP_KEYS, JWT_SECRET, etc.
- [ ] Strapi CORS configured
- [ ] Strapi running on localhost:1337
- [ ] Admin user created

## Content Setup
- [ ] Property content type created in Strapi
- [ ] All Property fields added (title, description, price, location, etc.)
- [ ] Public permissions configured for Property (find, findOne)
- [ ] First property created and published

## Frontend Development
- [ ] Strapi client utility created (lib/strapi.ts)
- [ ] Property card component created
- [ ] Home page updated to show properties
- [ ] Property detail page created
- [ ] Next.js image config updated
- [ ] Frontend displays properties correctly

## GitHub Setup
- [ ] .gitignore configured
- [ ] README.md created
- [ ] Code committed
- [ ] GitHub repository created
- [ ] Code pushed to GitHub

## Deployment
- [ ] Next.js deployed to Vercel
- [ ] Strapi deployed (Railway/Render/Heroku or local)
- [ ] Vercel environment variables configured
- [ ] Production URLs working

## Testing
- [ ] Property visible on Vercel deployment
- [ ] Property detail page working
- [ ] Images displaying correctly
- [ ] All links functional
